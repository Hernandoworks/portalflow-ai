# PortalFlow Core Blueprint

PortalFlow is an AI Agent Control Plane.

Architecture:

ChatGPT Custom GPT
-> PortalFlow Gateway
-> Permission Engine
-> Connector Registry
-> External Applications

Core modules:

- Command Router
- Connector SDK
- Agent Runtime
- Permission Engine
- Audit Engine
- Registry Layer

Principles:

1. Search before create
2. Plan before execute
3. Approve risky actions
4. Log every external change
